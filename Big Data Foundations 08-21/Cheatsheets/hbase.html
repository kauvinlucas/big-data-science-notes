Hbase cheatsheet



Start Hbase shell: hbase shell

Status: status

Hbase version: version

Table commands help: table_help

Info about user: whoami

See help: help <command>

Create table: create ‘<table name>’, {NAME=>’<family name>’}, {NAME=>’<family name>’}, …

create ‘<table name>’, ‘<family>’, ‘<family>’, … # by default, sqoop generates a txt file upon import (fields separated by comma and lines separated by line break)

List tables: list

Describe table: describe ‘<table name>’

Delete table: drop ‘<table name>’ #table must be disabled

drop_all <expression> # for example: drop_all ‘r.*’ (drop all tables starting with ‘r’)

Table verification: exists ‘<table name>’

scan ‘<table name>’

is_disabled ‘<table name>’

is_enable ‘<table name>’

Disable table: disable ‘<table name>’

disable_all ‘<expression>’

truncate ‘<table name>’ # will first disable table and then create a new table in place

Enable table: enable ‘<table name>’

Insert data: put ‘<table name>’,‘<key name>’,’<family name>:<column name>’,’<value>’

# to alter data, the same command must be issued with the same key a with a different value

Enable table: enable ‘<table name>’

Select table: get ‘<table name>’,’<key name>’ #get all entries by key

get ‘<table name>’,’<key name>’,’{COLUMNS=>[‘<family name 1>’,‘<family name 2>’,...]}’

get ‘<table name>’,’<key name>’,’{COLUMNS=>[‘<family name>:<column name>’,‘<family name>:<column>’,…]}’

scan ‘<table name>’ # select all of the rows of a table

scan ‘<table name>’, {COLUMNS=>[‘<family name>’]}

scan ‘<table name>’, {COLUMNS=>[‘<family name>:<column name>’]}

scan ‘<table name>’,{COLUMNS=>[‘<family name>’], VERSIONS=><number>}

scan ‘<table name>’,{STARTROW=>’<value>’, COLUMNS=>[‘<family name>’]} # select all the rows in a table which key starts with a value

Delete entries in table: delete ‘<table name>’,’<key name>’,’<family name>:<column name>’

delete ‘<table name>’,’<key name>’,’<family name>’ # delete of entries in the family

deleteall ‘<table name>’,’<key name>’ # delete of entries in a key

Alter table schema: alter ‘<table name>’,{NAME=>’<family name>’, VERSIONS=><version number>} # alter number of versions

alter ‘<table name>’,’delete’=>’<family name>’ # delete a family

Count table: count ‘<table name>’ # count keys in a table
