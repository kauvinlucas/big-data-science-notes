Redis cheatsheet



Strings:

Set string value: SET <key> <value>

SET <key> <value> XX # Replace existing value if exists (default option)

SET <key> <value> NX # Will fail if value exists

MSET <key1> <value> <key2> <value> <key3> <value> ... # Set value to several keys

Get string value: GET <key> # Returns (nil) if does not exists

MGET <key1> <key2> <key3> ... # Retrieve value from several keys

Get string length: strlen <key>

If a string is a number: decr <key> # Decrement value by 1

incr <key> # Increment value by 1

decrby <key> <increment> # Decrement value by a number

incrby <key> <decrement> # Increment value by a number



Key options:

Verify if key exists: exists <key> # Returns 0 if false, 1 if true

Delete existing key: del <key>

Get key type: type <key>

Define key expiration: expire <key> <seconds> # Only works if key exists

pexpire <key> <miliseconds>

set <key> <value> <seconds> ex # Creates a key and set expiration

set <key> <value> <miliseconds> px

Verify key expiration: ttl <key> # Returns time remaining in seconds, non existing key with return value of -2

pttl <key> # Returns time in miliseconds

Revoke key expiration: persist <key> # Must return -1 when using ttl

Basic configurations: config get <config parameter> # Get value of a configuration parameter

config get * # Get all configurations

config set
