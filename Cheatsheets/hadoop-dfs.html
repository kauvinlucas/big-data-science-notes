HDFS Distributed File System cheatsheet



Base command: hdfs dfs -<command>

Create directory: hdfs dfs -mkdir <directory>

hdfs dfs -mkdir -p <directory>/<directory>/<directory> # create folder structure

List directory: hdfs dfs ls <directory name>

hdfs dfs ls -R <directory name>

Remove directory: hdfs dfs rm <source>

hdfs dfs rm -r <source> #Delete directory

hdfs dfs rm -skipTrash <source> #Delete permanently

Copy/paste file or folder from local machine: hdfs dfs -put <source> <destination>

hdfs dfs -put -f <src> <destination> #Replace destiny if exists

hdfs dfs -put -l <replication value> <source> <destination> #Force replication factor

hdfs dfs -copyFromLocal <source> <destination>

Move file or folder from local: hdfs dfs -put <source> <destination> # (and remove file/folder from source)

hdfs dfs -moveFromLocal <source> <destination>

Receive file or folder from hdfs: get <source> <destination>

getmerge <source> <destination> #Creates a merged file from many sources

Move file or folder to local: hdfs dfs -get <source> <destination> # (and remove file/folder from source)

hdfs dfs -moveToLocal <source> <destination>

Move and copy files inside hdfs: hdfs dfs -cp <source> <destination> #Copy files

hdfs dfs -cp -f <source> <destination> #Replace destiny if exists

hdfs dfs -mv <source> <destination>

hdfs dfs -mv <source 1> <source 2> <source 3> ... <destination> #Move from several sources

Information:

Show disk usage: hdfs dfs -du -h <directory>

Show available disk space: hdfs dfs -df -h <directory>

Show directory stats: hdfs dfs -stat %<argument> <directory>

Count directory elements: hdfs dfs -count -h <directory>

Empty recycle bin: hdfs dfs -expunge

See file contents: cat <file>

Change replication factor: setrep <factor> <file>

Create file with 0KB of data: touchz <directory>

Check file integrity: checksum <file>

Show last 1KB of contents: tail <file>

Show first n lines of contents (Hadoop 2): hdfs dfs -cat <file> | head -n <number>

Find folder/file by expression: hdfs dfs -find <path> <expression> -print # example: hdfs dfs -find /folder -name \*.txt -print # print command is optional
