MongoDB cheatsheet



Create database: use <db name>

List current database: db

List all databases: show dbs

show databases

Create collection: db.createCollection(‘<name>’)

List all collections: show collections

Delete a collection: db.<name>.drop()

Drop database: db.dropDatabase()

Rename a collection: db.<name>.renameCollection(‘<new name>’)

Insert a document: db.<collection name>.insertOne(<BSON document>)

Insert more than one document: db.<collection name>.insertMany([<BSON document1>, <BSON document2>, …])

#Examples:

db.cliente.insertOne{nome: “Lucas”, idade: 20, conhecimento: “Windows, Linux”}

db.cliente.insertOne{nome: “Lucas”, idade: 20, conhecimento: [“Windows”, “Linux”]}

db.cliente.insertOne{nome: “Lucas”, idade: 20, conhecimento: [“Windows”, “Linux”], endereco: {rua: “Eusebio Matoso”, numero: 1375, bairro: “Pinheiro”}}

List documents in a collection: db.<collection name>.find().pretty()

Query document: db.<collection name>.find({<attribute>:<value>})

db.<collection name>.find({“<attribute>.<embed attribute>”:<value>}) #

db.<collection name>.find({<attribute>:{<operator>:<value>}}) #Common operators: $eq (equal to), $ne (different than), $gt (greater than), $gte(greater than or equal), $lt (less than), $lte (less than or equal), $in (in list), $nin (outside list), $not (not)

db.<collection name>.find({<attribute>:<value>}, {<attribute>:<projection value>}) #Incorporate projection on query. Projection value: 0 to not show attribute, 1 to show attribute.

See replica set configuration: rs.conf()

See replica set status: rs.status()
