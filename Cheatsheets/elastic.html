ElasticSearch CRUD operations:



HEAD: curl -XHEAD -v http://localhost:<port to kibana>/<index>/_doc/1 # Verify if document exist

HEAD <index>/_doc/1 {”<attribute>”:<value>, ...} # Command inside Dev Tools in Kibana

PUT: PUT <index>/_doc/<_id> {”<attribute>”:<value>, ...}# Reindex document (generate a new version)

PUT <index>/_create/<_id> {”<attribute>”:<value>, ...}# Creates a new document if it does not exist, if does exist throws an error

POST: POST <index>/_update/<_id> {”<attribute>”:<value>, …} # Update an existing document

POST <index>/_update/<_id> {“doc”:{”<attribute>”:<value>, …}} # Create a new document and generates an id

DELETE: DELETE <index>/_doc/<_id> # Delete a document

DELETE <index> # Delete an index

GET: GET <index>/_search # Search a index

GET <index>/_doc/<_id> # Search a index by id, includes attributes inserted by elastic such as _version

GET <index>/_source/<_id> # Search a index by id

GET <index>/_count # Count number of documents in indexes
